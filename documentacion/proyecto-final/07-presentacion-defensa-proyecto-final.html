Proyecto final

appTrackingPaquetesAPA controla paquetes internos con trazabilidad completa.

Sistema web cliente-servidor para Almacén Pájaro Azul: registra paquetes, genera guía, muestra seguimiento, permite reportar problemas y adjuntar comprobantes.

AngularFrontend para usuarios operativos, bodega, supervisión y gerencia.
ExpressBackend TypeScript con API REST, JWT, roles y validaciones.
MongoDBPersistencia documental para paquetes, tracking, problemas y comprobantes.
Ing. Alexander VarelaAvance 3 | Defensa final
Problema

El control manual dificulta saber dónde está cada paquete y quién debe responder.

Información dispersa

Sin guía centralizada, cada área consulta por mensajes o llamadas.

Poca trazabilidad

No siempre queda claro cuándo cambió el estado o quién atendió el paquete.

Sin respaldo ordenado

Los comprobantes o fotos pueden perderse si no se asocian al paquete.

Necesidad del cliente2
Solución

El sistema concentra el ciclo completo del paquete desde el registro hasta la revisión gerencial.

1Registrar paquete o varios paquetes juntos.
2Generar guía automática con formato APA-000001.
3Consultar detalle y seguimiento por guía.
4Reportar problemas y subir comprobantes.
5Revisar dashboard, reportes y auditoría.

Roles

Los permisos separan administración, operación y traslado.

Administrador

Gestiona usuarios, lugares, estados, paquetes, reportes y auditoría.

Usuario

Registra y consulta paquetes según los permisos de operación.

Motorista

Consulta paquetes asignados y registra avances de entrega.

Funcionalidades por rol6
Tiempo real

Socket.IO permite actualizar pantallas abiertas cuando cambia la operación.

  • Eventos para dashboard, paquetes, tracking, problemas, comprobantes, catálogos y auditoría.
  • Rooms por usuario, rol, paquete y guía para enviar avisos a quien corresponde.
  • Redis ayuda como adapter cuando está disponible; si no, el backend conserva funcionamiento local.
