# Entrega Proyecto Final - appTrackingPaquetesAPA

Esta carpeta contiene la documentación necesaria para la entrega final del proyecto, organizada según los requisitos del documento `Proyecto Final_861.pdf`.

## Archivos principales

| Archivo | Uso |
| --- | --- |
| `01-informe-digital-proyecto-final-appTrackingPaquetesAPA.pdf` | Informe digital final en PDF. Integra información general, justificación, alcance, beneficios, arquitectura, base de datos, seguridad, pruebas y conclusiones. |
| `02-manual-tecnico-appTrackingPaquetesAPA.pdf` | Manual técnico. Incluye tecnologías, plataforma, hardware, instalación, respaldo, arquitectura, backend, frontend, base de datos, pruebas y recomendaciones. |
| `03-manual-usuario-appTrackingPaquetesAPA.pdf` | Manual de usuario con capturas reales y lenguaje claro para bodega, operación, supervisión y gerencia. |
| `04-diagrama-base-datos-appTrackingPaquetesAPA.pdf` | Diagrama de colecciones de MongoDB listo para imprimir en una página. |
| `04-diagrama-base-datos-appTrackingPaquetesAPA.png` | Versión imagen del diagrama de base de datos. |
| `05-matriz-cumplimiento-avance-3.md` | Matriz que relaciona cada requisito del Avance 3 con la evidencia del repositorio. |
| `06-anexo-avances-1-y-2-integrados.md` | Integración de objetivos, alcance, requerimientos, casos de uso, glosario, colecciones y esquemas de los avances anteriores. |
| `07-presentacion-defensa-proyecto-final.pdf` | Presentación ejecutiva para defensa del proyecto. |
| `07-presentacion-defensa-proyecto-final.html` | Fuente HTML de la presentación. |
