# Backend de appTrackingPaquetesAPA

API backend para el sistema de trazabilidad de envíos internos de Almacén Pájaro Azul.

## Tecnologías

- Node.js
- Express
- TypeScript
- MongoDB
- Mongoose
- Redis
- dotenv
- cors
- helmet
- morgan
- bcryptjs
- jsonwebtoken

## Comandos

Instalar dependencias:

```bash
cd backend
npm install
```

Ejecutar en modo desarrollo:

```bash
npm run dev
```

Compilar para producción:

```bash
npm run build
```

Iniciar versión compilada:

```bash
npm run start
```

## Variables de entorno

Copiar `.env.example` a `.env` y configurar los valores locales.

## Puerto

El backend se expone en `http://localhost:3090`.
