# Frontend appTrackingPaquetesAPA

Aplicacion Angular para operar el sistema de tracking.

## Comandos

```bash
npm install
npm start
npm run build
```

URL local: `http://localhost:4300/login`

## Pantallas

- Login visual alineado con appTallerAPA.
- Dashboard.
- Paquetes.
- Nuevo paquete.
- Detalle de paquete.
- Tracking.
- Incidencias.
- Evidencias con upload real.
- Usuarios.
- Lugares.
- Estados.

## Conexion API

El proxy de desarrollo redirige `/api` al backend. Usar backend en `PUERTO=3090` para coincidir con la configuracion final.

## UX

Las pantallas internas incluyen estados de carga, mensajes de exito/error, formularios deshabilitados durante guardado y confirmaciones antes de eliminar/desactivar registros.
